Register Admin User

Warning: This page is for development use only. It allows you to register an email address as an admin user.

Important RLS Policy Information

If you're getting authorization errors when trying to access the admin interface, it's likely due to Supabase Row Level Security (RLS) restrictions.

To fix this, run the following SQL in your Supabase SQL Editor:

ALTER POLICY "admin_users_policy" ON "public"."admin_users" TO public USING (true);

This will allow the server to read admin user records without authentication.
